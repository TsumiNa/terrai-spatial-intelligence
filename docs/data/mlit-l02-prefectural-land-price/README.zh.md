# 国土交通省 L02 都道府县地价调查（2025）
[English](README.md) | [日本語](README.ja.md) | [中文](README.zh.md)

## 数据内容
2025-07-01 基准点价格、同比及土地/用途字段。通常单位为日元/m²，林地为日元/10a。获取埼玉、千叶、东京、神奈川四都县。
## 来源
[L02-2025 官方页](https://nlftp.mlit.go.jp/ksj/gml/datalist/KsjTmplt-L02-2025.html)
## 在本项目中的使用
作为成本和市场变化的第二套官方可比序列。API 键 `prefecturalLandPrice`，由 `mlit` 更新。
