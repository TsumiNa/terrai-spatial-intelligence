# MLIT A53 Multi-stage Inundation Assumptions (2025)
[English](README.md) | [日本語](README.ja.md) | [中文](README.zh.md)

## Data description
Flood-depth scenario polygons for multiple frequencies such as 1/10, 1/30, 1/50 and 1/100. A53 scenario/water-system fields, source layer and timestamps are retained. Acquired from the Kanto regional block and clipped to the mainland-Kanto window.
## Source
[A53-2025 official page](https://nlftp.mlit.go.jp/ksj/gml/datalist/KsjTmplt-A53-2025.html)
## Use in this project
Frequency-sensitive stress tests across roads, facilities and solar assets. API key `multistageFlood`; `mlit` task refresh.

## Commercial-use cautions
MLIT states that A53 differs from statutory Flood Control Act inundation maps. Use for comparative screening, not evacuation or final design.
