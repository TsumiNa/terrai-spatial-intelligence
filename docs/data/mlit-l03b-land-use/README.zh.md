# 国土交通省 L03-b 土地利用细分网格（2021）
[English](README.md) | [日本語](README.ja.md) | [中文](README.zh.md)

## 数据内容
2021年全国100米土地利用网格，含网格码、类别和卫星影像日期；按关东本土窗口裁剪 8 个一级网格（5238、5239、5240、5338、5339、5340、5439、5440）。
## 来源
[L03-b-2021 官方页](https://nlftp.mlit.go.jp/ksj/gml/datalist/KsjTmplt-L03-b-2021.html)
## 在本项目中的使用
作为用地适配、城市背景、道路环境和稀疏模型特征的可解释基础。API 键 `landUseMesh`，由 `mlit` 更新。
