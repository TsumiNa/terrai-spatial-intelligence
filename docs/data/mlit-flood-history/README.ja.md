# 国土交通省 全期間の水害GIS
[English](README.md) | [日本語](README.ja.md) | [中文](README.zh.md)

## データの内容
1896～2019年の記録水害ポリゴン（2025年3月までに公開）。日付・災害名・出典を保持して関東本土窓（東京・神奈川・千葉・埼玉）を抽出します。記録がないことは無危険を意味しません。
## 出典
[国土調査ダウンロード](https://nlftp.mlit.go.jp/kokjo/inspect/landclassification/download.html)
## 本 project での利用
道路、施設、太陽光の独立した被災履歴根拠。APIキーは`floodHistory`、`mlit`タスクで更新します。
