# GSI DEM5A

[English](README.md) | [日本語](README.ja.md) | [中文](README.zh.md)

- FL 状态：已接入
- 发布者：日本国土地理院（GSI）
- 原生分辨率：5 m

## 数据内容

- **格式** — GSI 基盘地图信息 DEM 系列的 GML 高程瓦片。本项目不随仓库分发原始 GML，而是把由其导出的数值作为属性存放在本地 GeoJSON 成果中。
- **分辨率** — DEM5A，5 m 网格间距。高程为平均海面（T.P.）以上的米值。
- **坐标系** — 来源为 JGD2011 地理坐标；所有派生成果统一写为 WGS84 GeoJSON（EPSG:4326）。
- **本地缓存范围** — 仅两个演示矩形：横滨保土谷（139.5835–139.5935 E, 35.4426–35.4504 N）与茂原（140.2757–140.2913 E, 35.4387–35.4513 N）。
- **项目读取的字段** — `elevation_m`（m）、`slope_deg`（度）、`local_relief_m`（m）、`low_point_m`（m），以及记录每个数值背后 DEM 产品的 `dem_source`。横滨全部 2,128 栋建筑当前均为 `dem_source = "DEM5A"`。
- **数据量** — 为 2,128 个建筑面、272 条道路区段、70 个茂原候选单元附加地形属性。
- **已知缺失与限制** — DEM5A 仅覆盖已实施航空激光测量的区域，其余地区回退到更粗产品，这正是逐要素保存 `dem_source` 的原因。植被、构筑物与测量误差使其并非纯粹的裸地面模型；以度表示的坡度是地形测量值，不是边坡稳定性结论。

## 来源

数据来自 GSI 基盘地图信息数值高程模型 DEM5A。下载需要免费注册；官方说明：https://service.gsi.go.jp/kiban/app/help/

## 在本项目中的使用

用于横滨和茂原的高程、坡度、局部起伏、低点代理，以及建筑和道路的地形暴露。确定性坡度/起伏计算仍属于 FL，不等于灾害概率或边坡稳定结论。本地产物包括 `data/yokohama/building_risk.geojson`、`road_priority.geojson` 和光伏网格中的地形字段。
