# MLIT W05 河川数据评估

[English](README.md) | [日本語](README.ja.md) | [中文](README.zh.md)

## 决定

W05 不接入 TerrAI。仓库不保留下载器、本地产物、API 数据键、自动任务或面向客户的河流图层。

## 数据内容

旧版数据提供神奈川、千叶2008年 JGD2000、约1:25,000河流中心线和相关点。与本次 MLIT FL 数据包中的其他图层相比，其年代明显较旧。

## 来源与条件

- [W05 官方页](https://nlftp.mlit.go.jp/ksj/gml/datalist/KsjTmplt-W05.html)
- 旧版官方页面明确标为**非商业使用**。

## 排除原因

TerrAI 面向商业客户演示。即使技术上隔离本地缓存，也容易形成对不可产品化图层的隐性依赖。只保留评估文档，既保存选源知识，也不会形成影子接入。

## 日后重新评估

接入河川数据前，应选择商业和再分发权利明确的最新来源，并比较覆盖、几何、更新周期、水文属性、法定权威以及与道路韧性、洪水和光伏分析的兼容性。新来源应重新经过正常 FL 时间戳、溯源、文档和 API 审核，而不是默认恢复已删除的 W05 任务。
