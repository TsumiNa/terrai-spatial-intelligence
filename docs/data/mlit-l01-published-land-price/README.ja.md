# 国土交通省 L01 地価公示（2026）
[English](README.md) | [日本語](README.ja.md) | [中文](README.zh.md)

## データの内容
2026-01-01時点の標準地。価格（円/m²）、面積、用途、用途地域等のL01原属性を保持します。埼玉・千葉・東京・神奈川の4都県を取得します。
## 出典
[L01-2026公式ページ](https://nlftp.mlit.go.jp/ksj/gml/datalist/KsjTmplt-L01-2026.html)
## 本 project での利用
太陽光・レジリエンス投資の比較地価と用地費背景。APIキー`publishedLandPrice`、`mlit`で更新。
