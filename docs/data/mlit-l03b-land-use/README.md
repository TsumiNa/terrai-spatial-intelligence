# MLIT L03-b Detailed Land-use Mesh (2021)
[English](README.md) | [日本語](README.ja.md) | [中文](README.zh.md)

## Data description
National 100 m mesh land-use polygons for 2021, including mesh code, land-use class and imagery date. TerrAI clips first-level meshes 5339 and 5340.
## Source
[L03-b-2021 official page](https://nlftp.mlit.go.jp/ksj/gml/datalist/KsjTmplt-L03-b-2021.html)
## Use in this project
Interpretable base for land compatibility, urban context, route surroundings and sparse-model features. API key `landUseMesh`; `mlit` refresh.

## Commercial-use cautions
100 m cells contain mixed land uses and are not parcel boundaries or current observations; retain imagery/vintage metadata.
