# 国土交通省 A53 多段階浸水想定（2025）
[English](README.md) | [日本語](README.ja.md) | [中文](README.zh.md)

## データの内容
1/10、1/30、1/50、1/100等の発生頻度別浸水深ポリゴン。水系・シナリオ属性、原レイヤ、時刻を保持します。関東地方整備局ブロックを取得し、関東本土窓で抽出します。
## 出典
[A53-2025公式ページ](https://nlftp.mlit.go.jp/ksj/gml/datalist/KsjTmplt-A53-2025.html)
## 本 project での利用
道路、施設、太陽光の頻度別ストレステスト。APIキー`multistageFlood`、`mlit`で更新。
