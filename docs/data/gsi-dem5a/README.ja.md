# GSI DEM5A

[English](README.md) | [日本語](README.ja.md) | [中文](README.zh.md)

- FL 状態：接続済み
- 公開者：国土地理院（GSI）
- 原生解像度：5 m

## データの内容

- **形式** — GSI 基盤地図情報 DEM シリーズの GML 標高タイル。本 project は生の GML を同梱せず、そこから導出した値をローカル GeoJSON 成果物の属性として保持します。
- **解像度** — DEM5A、5 m グリッド間隔。標高は平均海面（T.P.）からのメートル値です。
- **CRS** — 出典は JGD2011 地理座標。派生成果物はすべて WGS84 GeoJSON（EPSG:4326）で書き出します。
- **ここでキャッシュしている範囲** — 2 つのデモ矩形のみ：横浜・保土ケ谷（139.5835–139.5935 E, 35.4426–35.4504 N）と茂原（140.2757–140.2913 E, 35.4387–35.4513 N）。
- **project が読むフィールド** — `elevation_m`（m）、`slope_deg`（度）、`local_relief_m`（m）、`low_point_m`（m）、および各値の由来 DEM 製品を記録する `dem_source`。横浜の 2,128 建物はすべて現在 `dem_source = "DEM5A"` です。
- **データ量** — 建物ポリゴン 2,128 件、道路区間 272 件、茂原の候補セル 70 件に地形属性を付与。
- **既知の欠測と留意点** — DEM5A は航空レーザ測量が実施された範囲にのみ存在し、それ以外はより粗い製品で補われるため、`dem_source` を地物ごとに保持しています。植生・構造物・測定誤差があるため純粋な地盤面モデルではなく、度で表した傾斜は地形の測定値であって斜面安定性の結論ではありません。

## 出典

GSI 基盤地図情報数値標高モデル DEM5A。download には無料登録が必要です。https://service.gsi.go.jp/kiban/app/help/

## 本 project での利用

横浜・茂原の標高、傾斜、局所起伏、低所代理、建物/道路地形 exposure に利用します。決定論的な傾斜/起伏計算は FL であり、災害確率や斜面安定判断ではありません。`data/yokohama/building_risk.geojson`、`road_priority.geojson`、solar cell の地形 field に派生値があります。
