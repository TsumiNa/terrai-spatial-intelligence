# MLIT A56 Embankment Regulation Zones (2025)
[English](README.md) | [日本語](README.ja.md) | [中文](README.zh.md)

## Data description
Polygons for residential-land development and specified embankment regulation zones, source date 2025-07-18, with A56 attributes retained. Acquired for Saitama, Chiba, Tokyo and Kanagawa.
## Source
[A56-2025 official page](https://nlftp.mlit.go.jp/ksj/gml/datalist/KsjTmplt-A56-2025.html)
## Use in this project
Regulatory-screen evidence for slope work, roads and solar sites. API key `embankmentRegulation`; `mlit` refresh.
