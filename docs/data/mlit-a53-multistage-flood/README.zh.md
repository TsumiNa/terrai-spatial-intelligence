# 国土交通省 A53 多阶段淹水情景（2025）
[English](README.md) | [日本語](README.ja.md) | [中文](README.zh.md)

## 数据内容
提供1/10、1/30、1/50、1/100等频率的水深情景面，保留水系、情景、源图层和时间戳。获取関東地方整備局整块并按关东本土窗口裁剪。
## 来源
[A53-2025 官方页](https://nlftp.mlit.go.jp/ksj/gml/datalist/KsjTmplt-A53-2025.html)
## 在本项目中的使用
用于道路、设施和光伏资产的分频率压力测试。API 键 `multistageFlood`，由 `mlit` 更新。
