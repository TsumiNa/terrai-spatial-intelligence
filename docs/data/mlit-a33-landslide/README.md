# MLIT A33 Landslide Disaster Warning Zones (2025)
[English](README.md) | [日本語](README.ja.md) | [中文](README.zh.md)

## Data description
Polygons for warning/special-warning zones covering steep-slope collapse, debris flow and landslide hazards; source date 2025-08-01. Original A33 fields and provenance are retained.
## Source
[A33-2025 official page](https://nlftp.mlit.go.jp/ksj/gml/datalist/KsjTmplt-A33-2025.html)
## Use in this project
Direct exclusion/exposure evidence for slope, road, facility and solar analyses. API key `landslideWarning`; `mlit` task refresh.
