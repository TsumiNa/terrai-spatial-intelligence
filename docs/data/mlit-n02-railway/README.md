# MLIT N02 Railway Data (2025)
[English](README.md) | [日本語](README.ja.md) | [中文](README.zh.md)

## Data description
Railway-section and station geometries current at 2025-12-31, with operator/line/station attributes. TerrAI uses the UTF-8 release and clips both contexts.
## Source
[N02-2025 official page](https://nlftp.mlit.go.jp/ksj/gml/datalist/KsjTmplt-N02-2025.html)
## Use in this project
Access, evacuation redundancy and logistics context. API key `railway`; rebuilt by `mlit`.
