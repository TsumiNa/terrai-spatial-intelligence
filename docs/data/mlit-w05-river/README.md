# MLIT W05 River Data (restricted)
[English](README.md) | [日本語](README.ja.md) | [中文](README.zh.md)

## Data description
2008 JGD2000 river centreline and related point data at approximately 1:25,000 for Kanagawa and Chiba.
## Source
[W05 official page](https://nlftp.mlit.go.jp/ksj/gml/datalist/KsjTmplt-W05.html)
## Use in this project
Optional local evaluation only. Run `uv run python -m terrai_spatial data update --only mlit_river`; output is Git-ignored and absent from the API.
## License
The official legacy page explicitly marks the dataset **non-commercial**.
## Commercial-use cautions
It is not part of the customer-facing FL. A commercially licensed/current river source must replace it before product use.
