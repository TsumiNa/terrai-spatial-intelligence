# TerrAI Documentation Index

[中文](README.md) | [日本語](README.ja.md) | [English](README.en.md)

`docs/` uses only the following five categories. No documents other than this trilingual index belong directly under `docs/`.

| Directory | Content |
|---|---|
| [`architecture/`](architecture/FRONTEND_BACKEND.en.md) | Current system boundaries, component responsibilities, and call structure |
| [`refactor/`](refactor/fl-sl-al-platform/00-overview.en.md) | Context, rationale, and concrete PR-staged plans for each refactor |
| [`data/`](data/README.en.md) | Source, project use, license, and commercial cautions for every integrated FL dataset |
| [`summary/`](summary/2026-07-prototype-state.en.md) | Functional validation, evaluation summaries, and important non-refactor project decisions |
| [`others/`](others/) | Text that genuinely fits none of the four categories above; explain why it belongs here |

Every document is maintained as a Chinese `.md`, Japanese `.ja.md`, and English `.en.md` group. See [repository-doc-boundaries.instructions.md](../.github/instructions/repository-doc-boundaries.instructions.md) for the complete rules.
