# 国土交通省全期间水害 GIS
[English](README.md) | [日本語](README.ja.md) | [中文](README.zh.md)

## 数据内容
收录1896–2019年洪灾事件面，发布更新截至2025年3月。保留日期、灾害名和来源并裁剪关东本土窗口（东京、神奈川、千叶、埼玉）；无记录不代表无风险。
## 来源
[国土调查下载](https://nlftp.mlit.go.jp/kokjo/inspect/landclassification/download.html)
## 在本项目中的使用
为道路、设施和光伏提供独立历史灾害证据。API 键 `floodHistory`，由 `mlit` 任务更新。
