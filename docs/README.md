# TerrAI Documentation Index

`docs/` uses only the following five categories. No document other than this index belongs directly under `docs/`.

| Directory | Content | Language policy |
|---|---|---|
| [`architecture/`](architecture/FRONTEND_BACKEND.en.md) | Current system boundaries, component responsibilities, and call structure | Chinese, Japanese, and English |
| [`refactor/`](refactor/fl-sl-al-platform/00-overview.md) | Context, rationale, and concrete PR-staged plans for each refactor | English by default |
| [`data/`](data/README.en.md) | Source, project use, license, and commercial cautions for every integrated FL dataset | Chinese, Japanese, and English |
| [`summary/`](summary/2026-07-prototype-state.en.md) | Functional validation, evaluation summaries, and important non-refactor project decisions | Chinese, Japanese, and English |
| [`others/`](others/) | Text that genuinely fits none of the four categories above; explain why it belongs here | English by default |

Only `architecture/`, `data/`, and `summary/` require trilingual document groups. Root documents, this index, `refactor/`, and `others/` use English-only `.md` files unless a specific need justifies additional translations. See [repository-doc-boundaries.instructions.md](../.github/instructions/repository-doc-boundaries.instructions.md) for the complete rules.
