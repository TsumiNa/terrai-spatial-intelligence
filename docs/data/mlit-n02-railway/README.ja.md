# 国土交通省 N02 鉄道データ（2025）

## データの内容
2025-12-31時点の鉄道路線・駅形状と事業者、路線、駅属性。UTF-8版から両地域を抽出します。

[N02-2025公式ページ](https://nlftp.mlit.go.jp/ksj/gml/datalist/KsjTmplt-N02-2025.html)
## 本 project での利用
アクセス、避難冗長性、物流背景。APIキー`railway`、`mlit`で更新。
