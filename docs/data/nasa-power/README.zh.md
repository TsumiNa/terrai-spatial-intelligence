# NASA POWER 太阳辐照气候数据

[English](README.md) | [日本語](README.ja.md) | [中文](README.zh.md)

- FL 状态：已接入并缓存
- 指标：`ALLSKY_SFC_SW_DWN`
- 气候期：2001–2020

## 来源

NASA POWER Climatology API：https://power.larc.nasa.gov/docs/services/api/temporal/climatology/

## 在本项目中的使用

提供茂原市太阳能资源的区域长期背景，当前年均 GHI 约 1,378 kWh/m²。它只说明区域气候资源，不进入逐网格发电量模拟，也不能替代设备、遮挡、朝向、温度和损耗模型。

## 商业使用注意

商业报告应标明变量、气候期、空间尺度和下载时间。气候平均值不是项目级 P50/P90 发电预测，也不能用于融资级收益保证；若未来接入更细时间序列，需重新验证单位、缺测和偏差。
