# FL Dataset Catalog

[中文](README.md) | [日本語](README.ja.md) | [English](README.en.md)

This directory documents datasets already integrated into the TerrAI Foundation Data Layer (FL). Each card records source, project use, license, commercial-use cautions, local products, and refresh path.

| Dataset | Status | Card |
|---|---|---|
| GSI DEM5A | Integrated | [Terrain elevation](gsi-dem5a.en.md) |
| GSI maps and visual tiles | Integrated | [Standard/imagery/relief/slope](gsi-map-tiles.en.md) |
| Google Satellite Embedding V1 | Integrated | [Annual remote-sensing representation](google-satellite-embedding.en.md) |
| OpenStreetMap | Integrated | [Buildings, roads, and context](openstreetmap.en.md) |
| Yokohama regional disaster bases | Integrated | [Official public facilities](yokohama-disaster-bases.en.md) |
| NASA POWER | Integrated | [Solar climate context](nasa-power.en.md) |
| TEPCO public system information | Integrated, redistribution restricted | [Regional grid-capacity screen](tepco-grid.en.md) |

Confirmed but not integrated and excluded sources are summarized in [`docs/summary/open-data-landscape.en.md`](../summary/open-data-landscape.en.md). Machine-readable status is in `data/external/source_registry.json`.
