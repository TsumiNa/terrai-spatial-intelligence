# 国土交通省 L03-b 土地利用細分メッシュ（2021）
[English](README.md) | [日本語](README.ja.md) | [中文](README.zh.md)

## データの内容
2021年全国100m土地利用メッシュ。メッシュコード、土地利用種別、衛星写真年月日を含み、一次メッシュ8面（5238・5239・5240・5338・5339・5340・5439・5440）を関東本土窓で抽出します。
## 出典
[L03-b-2021公式ページ](https://nlftp.mlit.go.jp/ksj/gml/datalist/KsjTmplt-L03-b-2021.html)
## 本 project での利用
土地適合性、都市背景、沿道環境、疎データモデル特徴量の解釈可能な基礎。APIキー`landUseMesh`、`mlit`で更新。
